# TodoApp
Simple Todo app with unit testing

This is simple todo app where user add todo-task , update their task and delete their task.

**Installation**

For iOS first run

```
npx pod-install
```

For android you don't have do any extra steps.

For Run application:-

**Android:-**

run 
```
npx react-native run-android
```

**iOS:-**
```
npx react-native run-android
```


**Run test**

```
yarn run test
```
